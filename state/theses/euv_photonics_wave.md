# Thesis: EUV — Photonics/Optical Communication as Next AI Infrastructure Wave

## Date: 2026-05-15
## Status: ACTIVE (ambush position)

## Investment Logic

The AI infrastructure buildout follows sequential waves of bottlenecks:

1. **Wave 1 (past):** GPU compute — NVDA dominated, hyper-growth phase largely priced in
2. **Wave 2 (current):** High-performance memory — SNDK, HBM providers. Logic: exponential compute increase → machines need more memory to store intermediate results on the fly
3. **Wave 3 (next):** Optical communication / photonics — Logic: as data centers scale to gigawatt+, electrical interconnects become the bottleneck. Optical reduces transmission loss significantly, enables higher bandwidth between racks/clusters

## The Bet

EUV (Corgi Lithography & Semiconductor Photonics ETF) provides exposure to the photonics value chain — companies involved in light-based data transmission technologies that will be critical for next-gen AI data center interconnects.
